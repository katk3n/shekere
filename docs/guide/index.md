# What is Shekere?

Shekere is an audio-visualizer application that allows you to run JavaScript-based sketches within a desktop application built with Tauri and Rust.

## Features

- **JS Sketch Support**: Run your visuals using standard JavaScript.
- **Audio Reactive**: Real-time audio analysis data at your fingertips.
- **Modern Tech Stack**: React for the control panel, Rust for the backend.
