# What is Shekere?

Shekere is an audio-reactive visualizer desktop application that allows you to create and run stunning visuals using JavaScript. It bridges the gap between high-performance native technology and the flexibility of the web.

## Why Shekere?

In the world of creative coding and VJing, performance and ease of use often conflict. Shekere aims to provide a platform where you can write code using familiar web technologies without sacrificing the smoothness and system integration of a native application.

## Core Features

### 🚀 High Performance
Built on the **Tauri** framework with a **Rust** backend, Shekere is lightweight, fast, and consumes minimal system resources compared to traditional electron-based apps.

### 🎨 Familiar Web Technologies
Write your visuals using standard JavaScript or TypeScript. Leverage powerful web APIs like **Canvas 2D**, **WebGL**, or **Three.js** to bring your imagination to life.

### 🔊 Audio-Reactive by Default
Shekere comes with built-in support for **Meyda**, a powerful audio feature extraction library. Real-time audio data (RMS, frequency bands, spectral features, etc.) is automatically provided to your sketches, allowing your visuals to "dance" to the music instantly.

### ⚡ Live Sketching & Fast Iteration
Easily switch between different sketches or update your code to see results in real-time. Whether you are performing live at a club or experimenting in your studio, Shekere is built for speed.

## Target Audience

- **Creative Coders**: Developers looking for a dedicated environment to explore generative art and audio visualization.
- **VJs & Visual Artists**: Performers who need a reliable, high-performance tool to generate visuals that sync perfectly with live audio.
- **Students & Hobbyists**: Anyone interested in learning how to combine code and sound to create beautiful experiences.
