# Shekereとは

Shekere（シェケレ）は、JavaScriptを使って印象的なビジュアルを作成・実行できる、オーディオ・リアクティブなビジュアライザー・デスクトップアプリケーションです。ハイパフォーマンスなネイティブ技術と、ウェブ技術の持つ柔軟性を融合させています。

## なぜ Shekere なのか？

クリエイティブ・コーディングやVJの世界では、パフォーマンスと使いやすさがしばしば相反します。Shekereは、ネイティブアプリケーションの滑らかさとシステム連携を犠牲にすることなく、使い慣れたウェブ技術でコードを書けるプラットフォームを提供することを目指しています。

## 主な特徴

### 🚀 ハイパフォーマンス
**Tauri** フレームワークと **Rust** バックエンドを採用。従来のElectronベースのアプリと比較して非常に軽量かつ高速で、システムリソースの消費を最小限に抑えています。

### 🎨 使い慣れたウェブ技術
標準的な JavaScript や TypeScript でビジュアルを作成できます。**Canvas 2D**、**WebGL**、**Three.js** などの強力なウェブAPIを活用して、想像力を形にしましょう。

### 🔊 オーディオ・リアクティブ標準対応
強力な音声解析ライブラリである **Meyda** を標準サポートしています。音量(RMS)や周波数、スペクトル特徴などのリアルタイムデータが自動的にスケッチに提供されるため、すぐに音楽に連動するビジュアルを作成できます。

### ⚡ ライブ・スケッチングと迅速な反復
異なるスケッチの切り替えや、コードの更新をリアルタイムに反映できます。クラブでのライブパフォーマンスでも、スタジオでの実験でも、Shekereはスピードを重視して設計されています。

## 想定ユースケース

- **クリエイティブ・コーダー**: ジェネレーティブアートやオーディオビジュアライズを探求するための専用環境を求めている開発者。
- **VJ & ビジュアルアーティスト**: ライブオーディオと完璧に同期するビジュアルを生成するための、信頼性が高く高性能なツールを必要としているパフォーマー。
- **学生 & ホビーユーザー**: コードと音を組み合わせて美しい体験を作る方法を学びたいすべての人。
